一、 全局设计系统 (Design System)对标 Linear、Raycast 与 macOS 原生暗色模式，追求极致的“克制”与“深邃”。

色彩语义矩阵 (Color Tokens)
语义   色值 (Hex/RGBA)   使用场景
深渊背景   #0A0A0C   全局最底层背景，纯黑偏蓝，防 OLED 烧屏且深邃。

卡片悬浮   #16161A (Opacity 90%)   右侧面板、弹窗、气泡背景，带毛玻璃效果。

边界分割   rgba(255,255,255, 0.06)   所有卡片边框、分割线，绝不用纯灰线。

主行动点   #3B82F6 (科技蓝)   主按钮、高亮标签、进度条。

AI 幽灵态   #8B5CF6 (紫罗兰)   AI 生成的建议、未确认的草稿、呼吸灯高亮。

高危熔断   #EF4444 (警示红)   封号预警、高危词拦截、一键熔断按钮。

安全通行   #10B981 (翡翠绿)   iLink 连接正常、发送成功、价值评分高。

光影与层级 (Elevation & Shadows)
铁律：全局禁用 PyQt 原生 QGraphicsDropShadowEffect（会导致 CPU 飙升且边缘生硬）。
实现方案：
  层级 1 (贴地卡片)：无边框，仅靠 #16161A 背景色与深渊背景区分。
  层级 2 (悬浮面板/弹窗)：使用 Figma 导出的 24px 模糊半径、透明度 40% 的黑色 9 宫格 PNG 作为 border-image，实现完美的 macOS 级弥散阴影。
  层级 3 (Toast/Tooltip)：背景色提亮至 #27272A，加 1px 亮边 rgba(255,255,255, 0.1) 模拟物理高光。

动效物理引擎 (Motion Physics)
全局缓动曲线：摒弃线性动画，统一使用 cubic-bezier(0.25, 0.1, 0.25, 1.0) (类似 iOS 的弹簧曲线)。
标准时长：
  微交互 (Hover/点击反馈)：120ms
  面板滑出/收起 (右侧战术面板)：350ms
  状态渐变 (呼吸灯/颜色切换)：600ms

二、 核心布局架构 (Layout Architecture)

采用 “左-中-右”非对称三栏折叠布局，支持响应式拖拽。

左侧：极简导航轨 (Navigation Rail) - 宽 64px
视觉：背景 #0A0A0C，右侧带 1px 分割线。
交互：
  图标尺寸 24x24，未选中为 #71717A，选中为 #FFFFFF 且左侧出现 3px 宽的蓝色指示条。
  Hover 交互：鼠标悬停时，图标右侧 12px 处浮现黑底白字的 Tooltip（带 120ms 延迟，防误触）。

中置：微信物理映射舱 (WeChat Viewport) - 自适应宽度 (最小 400px)
视觉：这是 DWM 缩略图映射的微信真实窗口。
融合处理：微信原生是白底，为了融入暗色主题，我们在映射层上方覆盖一层 mix-blend-mode: multiply (正片叠底) 的暗色遮罩，或者在微信窗口外围加一圈 12px 的暗色圆角边框，使其看起来像是嵌在我们的系统里。
防穿透铁律：当右侧面板展开或系统弹窗时，中置舱必须覆盖一层 rgba(0,0,0, 0.4) 的遮罩，物理阻断用户对微信的直接点击，防止“双端打架”。

右侧：战术指挥面板 (Tactical Panel) - 宽 420px (可拖拽至 500px)
视觉：背景 #16161A，左侧带 1px 分割线。
交互：采用 QStackedWidget。未选中客户时，显示“数据看板”；选中客户时，伴随 350ms 的贝塞尔曲线滑出“客户 360 视图”。

三、 核心模块 UI 交互深度拆解

场景 A：私聊指挥舱与“幽灵卡片”交互 (核心亮点)当客户发来消息，AI 生成回复建议时，UI 如何表现？

消息流入：中置舱（微信）显示客户消息。同时，右侧面板底部弹出 AI 建议。
幽灵卡片 (Ghost Card) 设计：
   AI 建议不是直接放在输入框里，而是一张悬浮在输入框上方的“紫色半透明卡片”。
   打字机动效：文字以 30ms/字 的速度流式渲染，光标在末尾闪烁。
   策略注解：卡片底部用 12px 灰色字体显示：💡 策略：价值锚点转移 | 预估转化率：85%”。
采纳与拒绝交互：
   快捷键采纳：老板按下 Ctrl + Enter，卡片瞬间化为实体消息（变绿），通过 iLink 毫秒级发送，卡片消失。
   鼠标采纳：老板点击卡片上的“发送”图标，触发 120ms 的缩放点击反馈。
   修改后发送：老板点击卡片，卡片降落入输入框，变为可编辑状态。
   拒绝/忽略：老板不理会，5 秒后卡片透明度从 100% 渐变至 0%（600ms），自动消散。

场景 B：动态表单与“呼吸灯”回填 (数据联动)当 AI 从对话中提取到“预算：30万”时，右侧 CRM 表单如何更新？

触发：AI 返回的 JSON 中包含 {"budget": "300000"}。
定位：系统找到右侧面板“客户档案”卡片中的“预算”输入框。
呼吸灯动效 (重点)：
   输入框边框颜色从 rgba(255,255,255, 0.06) 突变为 #8B5CF6 (AI 紫)。
   背景色闪烁两次：rgba(139, 92, 246, 0.15) -> 透明 -> rgba(139, 92, 246, 0.15) -> 透明（总时长 1.2s）。
   数字“300,000”以数字滚动动效（从 0 滚到 300000，耗时 400ms）填入。
归档：动效结束后，输入框恢复初始样式，但右上角出现一个极小的“✨ AI 已更新” Tooltip，3 秒后消失。

场景 C：防打架机制的“琥珀色警告” (冲突处理)当老板在微信里手动打字，AI 准备发送消息时。

   当进度条走完，横幅向上收起消失，AI 恢复待命。

场景 D：公域作战室 (朋友圈/视频号) 的“异步视觉流”RPA 在后台刷朋友圈，UI 如何展示分析结果而不打扰老板？

   标题：“🎯 捕获商业触发器”。

四、 极端场景下的 UI 降级与反馈 (容错设计)

一个顶级的系统，在出错时的 UI 表现往往最能体现专业度。

AI 大模型超时 (8秒降级)
0-8秒：幽灵卡片显示“AI 思考中...”，带有三个紫色圆点交替跳动的 Loading 动效。
第 8 秒：如果主模型未返回， Loading 瞬间变为黄色警告图标“⚡ 主模型拥堵，已无缝切换至本地 Qwen”。
第 8.5 秒：本地模型生成结果，卡片继续以打字机效果渲染。整个过程不中断，不弹窗报错，仅做状态提示。

视频号/朋友圈 触达限流锁死

  “发送私信”按钮变为深灰色 (#3F3F46)，文字变为“今日额度已耗尽”。
  按钮下方出现一行红色小字：“🔒 距离额度重置还有 14:23:05”（实时跳动的倒计时）。
  物理锁死：按钮 setEnabled(False)，且 Hover 时鼠标指针变为“禁止”图标 (🚫)，彻底断绝老板手滑的可能。

动效推演：

  全局遮罩：整个软件界面在 200ms 内覆盖一层 rgba(239, 68, 68, 0.05) 的极淡红色遮罩。
  状态切换：所有自动化模块的指示灯（绿）瞬间熄灭，变为灰色。顶部状态栏出现常驻红字：“🚨 战术静默模式：所有 RPA 与自动发送已物理切断”。
  解除：再次点击按钮，红色遮罩如退潮般向上消散（300ms），系统恢复。

五、 给 UI/前端 工程师的“避坑军令状”

绝对禁止在主线程做动画：所有的倒计时、进度条、呼吸灯，必须使用 QPropertyAnimation 或 QTimer，严禁在循环里用 time.sleep() 或阻塞式计算，否则 UI 会卡成 PPT。
长列表虚拟滚动：线索资产库和聊天记录列表，当数据超过 500 条时，必须使用 QListView 配合自定义的 Model/Delegate 实现虚拟滚动 (只渲染可视区域的 Item)，严禁直接把几千个 Widget 塞进 QScrollArea，那会导致内存直接撑爆。
高分屏适配 (HiDPI)：在 main.py 最顶部必须加上 QApplication.setHighDpiScaleFactorRoundingPolicy 和 AA_EnableHighDpiScaling，否则在 4K 屏幕上，我们的 1px 边框和精致图标会糊成一团。

数据库底座：全局单线程写队列 (core/database.py)彻底封死 database is locked，实现读写分离与乐观锁基础。

core/database.py
import sqlite3
import queue
import threading
from PyQt6.QtCore import QObject, pyqtSignal

class DatabaseManager(QObject):
    """全局单线程数据库写入队列 (WAL模式)"""
    write_finished = pyqtSignal(bool, str) # 成功标志, 错误信息

    def init(self, db_path="waos_x.db"):
        super().init()
        self.db_path = db_path
        self.write_queue = queue.Queue()
        
        # 初始化 WAL 模式，提升并发读性能
        self._init_db()
        
        # 启动后台单线程写入循环
        self.worker_thread = threading.Thread(target=self._write_loop, daemon=True)
        self.worker_thread.start()

    def _init_db(self):
        conn = sqlite3.connect(self.db_path)
        conn.execute("PRAGMA journal_mode=WAL;")
        conn.execute("""
            CREATE TABLE IF NOT EXISTS crm_leads (
                id TEXT PRIMARY KEY,
                wx_account_hash TEXT NOT NULL,
                name TEXT,
                stage TEXT DEFAULT 'new',
                version INTEGER DEFAULT 1,
                updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
            )
        """)
        conn.commit()
        conn.close()

    def enqueue_write(self, sql: str, params: tuple):
        """将写任务丢入队列，绝不阻塞主线程"""
        self.write_queue.put((sql, params))

    def _write_loop(self):
        """后台单线程死循环，串行执行写操作"""
        conn = sqlite3.connect(self.db_path, check_same_thread=False)
        while True:
            sql, params = self.write_queue.get()
            try:
                conn.execute(sql, params)
                conn.commit()
                self.write_finished.emit(True, "")
            except Exception as e:
                self.write_finished.emit(False, str(e))
            finally:
                self.write_queue.task_done()

    def _start_typewriter(self):
        """30ms/字 的打字机动效"""
        self.timer = QTimer(self)
        self.timer.timeout.connect(self._type_next_char)
        self.timer.start(30)

        if self.current_index < len(self.full_text):
            self.text_label.setText(self.text_label.text() + self.full_text[self.current_index])
            self.current_index += 1

    def _find_and_bind_wechat(self):

        user32 = ctypes.WinDLL('user32')
        hwnd = user32.FindWindowW("WeChatMainWndForPC", None)
        
        if not hwnd:
            print("⚠️ 未找到运行中的微信主窗口，请先打开微信！")
            return

        self.wechat_hwnd = hwnd
        my_hwnd = int(self.winId()) # 获取 PyQt 控件的原生 HWND
        
        # 注册 DWM 缩略图
        res = dwmapi.DwmRegisterThumbnail(my_hwnd, self.wechat_hwnd, ctypes.byref(self.hThumbnail))
        if res == 0:
            print("✅ DWM 映射注册成功！")
            self.placeholder.hide()
            self.bind_btn.hide()
            self._update_thumbnail_props()
            self.resize_timer.start(100) # 每 100ms 检查一次尺寸变化
        else:
            print(f"❌ DWM 注册失败，错误码: {res}")

    def _update_thumbnail_props(self):
        """更新映射属性 (尺寸、可见性、透明度)"""
        if not self.hThumbnail: return
        
        props = DWM_THUMBNAIL_PROPERTIES()
        props.dwFlags = DWM_TNP_RECTDESTINATION | DWM_TNP_VISIBLE | DWM_TNP_OPACITY | DWM_TNP_SOURCECLIENTAREAONLY
        props.rcDestination = wintypes.RECT(0, 0, self.width(), self.height())
        props.fVisible = True
        props.opacity = 255
        props.fSourceClientAreaOnly = True # 只映射客户区，不映射微信原生标题栏
        
        dwmapi.DwmUpdateThumbnailProperties(self.hThumbnail, ctypes.byref(props))

    def resizeEvent(self, event):
        super().resizeEvent(event)
        if self.bind_btn.isVisible():
            self.bind_btn.move(self.width() - 160, 20)
        self._update_thumbnail_props()

    def closeEvent(self, event):
        """窗口关闭时释放 DWM 资源"""
        if self.hThumbnail:
            dwmapi.DwmUnregisterThumbnail(self.hThumbnail)
        super().closeEvent(event)

import hashlib
from PyQt6.QtWidgets import QInputDialog, QMessageBox

class AccountHashManager:
    """账号 SHA256 哈希隔离管理器"""
    
    @staticmethod
    def generate_hash(wx_id: str) -> str:
        """将微信号转换为不可逆的 SHA256 哈希"""
        if not wx_id or len(wx_id.strip()) < 3:
            raise ValueError("微信号格式不合法")
        return hashlib.sha256(wx_id.strip().encode('utf-8')).hexdigest()

    @staticmethod
    def bind_account_interactive(parent_widget=None) -> str | None:
        """弹窗让人工输入微信号并绑定 (首次启动时调用)"""
        wx_id, ok = QInputDialog.getText(
            parent_widget, 
            "🔐 账号物理隔离绑定", 
            "请输入当前登录的微信号 (如: wxid_xxx 或 自定义ID):n(此ID将作为底层数据的联合主键，绝不上传服务器)",
            text="wxid_test123"
        )
        
        if ok and wx_id:
            try:
                hash_val = AccountHashManager.generate_hash(wx_id)
                print(f"✅ 账号绑定成功！当前数据隔离哈希: {hash_val[:16]}...")
                return hash_val
            except ValueError as e:
                QMessageBox.warning(parent_widget, "绑定失败", str(e))
        return None

from paddleocr import PaddleOCR

class RightBrainReal(QThread):
    """右脑真实物理层：后台静默截图 + PaddleOCR 视觉感知"""
    def init(self, wechat_hwnd: int):
        super().init()
        self.wechat_hwnd = wechat_hwnd
        self.running = True
        # 初始化真实 OCR 引擎 (首次运行会自动下载模型)
        print("👁️ [右脑] 正在加载 PaddleOCR 模型 (这可能需要一分钟)...")
        self.ocr = PaddleOCR(use_angle_cls=True, lang="ch", show_log=False)

    def _capture_background_window(self):
        """核心魔法：使用 PrintWindow 截取被遮挡窗口的真实画面"""
        try:
            # 获取窗口尺寸
            left, top, right, bottom = win32gui.GetWindowRect(self.wechat_hwnd)
            width = right - left
            height = bottom - top

            # 创建设备上下文 (DC)
            hwnd_dc = win32gui.GetWindowDC(self.wechat_hwnd)
            mfc_dc = win32ui.CreateDCFromHandle(hwnd_dc)
            save_dc = mfc_dc.CreateCompatibleDC()
            
            # 创建位图对象
            bmp = win32ui.CreateBitmap()
            bmp.CreateCompatibleBitmap(mfc_dc, width, height)
            save_dc.SelectObject(bmp)
            
            # 核心 API：PrintWindow 支持后台/被遮挡截图 (PW_RENDERFULLCONTENT 保证 DWM 渲染完整)
            win32gui.PrintWindow(self.wechat_hwnd, save_dc.GetSafeHdc(), 3)
            
            # 转换为 OpenCV 格式 (numpy array)
            bmpinfo = bmp.GetInfo()
            bmpstr = bmp.GetBitmapBits(True)
            img = np.frombuffer(bmpstr, dtype=np.uint8)
            img = img.reshape((bmpinfo['bmHeight'], bmpinfo['bmWidth'], 4))
            img = cv2.cvtColor(img, cv2.COLOR_BGRA2BGR)
            
            # 清理资源
            win32gui.DeleteObject(bmp.GetHandle())
            save_dc.DeleteDC()
            mfc_dc.DeleteDC()
            win32gui.ReleaseDC(self.wechat_hwnd, hwnd_dc)
            
            return img
        except Exception as e:
            print(f"⚠️ [右脑] 截图失败: {e}")
            return None

    def run(self):
        print("✅ [右脑] 视觉感知引擎启动，开始后台巡逻...")
        while self.running:
            # 1. 后台静默截图 (即使微信在后台也能截到)
            frame = self._capture_background_window()
            
            if frame is not None:
                # 2. 送入 PaddleOCR 进行真实识别
                result = self.ocr.ocr(frame, cls=True)
                
                if result and result[0]:
                    for line in result[0]:
                        box, (text, confidence) = line
                        # 3. 寻找关键视觉锚点 (例如：发现“朋友圈”三个字)
                        if "朋友圈" in text and confidence > 0.8:
                            # 计算文字中心点坐标
                            center_x = int((box[0][0] + box[2][0]) / 2)
                            center_y = int((box[0][1] + box[2][1]) / 2)
                            
                            print(f"🎯 [右脑] 发现视觉锚点: '{text}' 坐标: ({center_x}, {center_y})")
                            # 通过事件总线广播坐标，供 RPA 鼠标模块点击
                            event_bus.vision_ui_element_found.emit("朋友圈入口", (center_x, center_y))
                            break
            
            # 视觉流不需要太高频，每秒 1-2 帧足够，防止 CPU 爆炸
            time.sleep(0.8) 

    def stop(self):

core/rpa_robotic_arm.py
import time
import random
import math
import win32api
import win32con
from PyQt6.QtCore import QThread
from core.event_bus import event_bus

class RoboticArm(QThread):
    """RPA 拟人机械臂 (负责执行物理点击)"""
    def init(self):
        super().init()
        # 监听右脑发来的坐标指令
        event_bus.cmd_click_coordinate.connect(self._execute_human_click)

    def _generate_bezier_curve(self, start, end, control1, control2, steps=30):
        """生成三阶贝塞尔曲线轨迹点"""
        points = []
        for i in range(steps + 1):
            t = i / steps
            # 贝塞尔曲线公式
            x = (1-t)**3 * start[0] + 3*(1-t)**2 * t * control1[0] + 3*(1-t) * t**2 * control2[0] + t**3 * end[0]
            y = (1-t)**3 * start[1] + 3*(1-t)**2 * t * control1[1] + 3*(1-t) * t**2 * control2[1] + t**3 * end[1]
            points.append((int(x), int(y)))
        return points

    def _execute_human_click(self, target_x: int, target_y: int):
        """执行拟人化点击"""
        # 1. 获取当前鼠标位置
        curr_x, curr_y = win32api.GetCursorPos()
        
        # 2. 生成随机控制点 (模拟人类手部的偏移和微颤)
        dist = math.hypot(target_x - curr_x, target_y - curr_y)
        offset = dist * 0.3
        
        ctrl1 = (curr_x + (target_x - curr_x)*0.3 + random.uniform(-offset, offset),
                 curr_y + (target_y - curr_y)*0.3 + random.uniform(-offset, offset))
        ctrl2 = (curr_x + (target_x - curr_x)*0.7 + random.uniform(-offset, offset),
                 curr_y + (target_y - curr_y)*0.7 + random.uniform(-offset, offset))
        
        # 3. 生成轨迹并移动
        points = self._generate_bezier_curve((curr_x, curr_y), (target_x, target_y), ctrl1, ctrl2)
        
        for px, py in points:
            win32api.SetCursorPos((px, py))
            # 模拟人类移动的非匀速 (两头慢，中间快)
            time.sleep(random.uniform(0.005, 0.02)) 
            
        # 4. 到达目标点后，随机延迟一下再点击 (人类反应时间)
        time.sleep(random.uniform(0.1, 0.3))
        
        # 5. 物理按下并抬起左键
        win32api.mouse_event(win32con.MOUSEEVENTF_LEFTDOWN, target_x, target_y, 0, 0)
        time.sleep(random.uniform(0.05, 0.15))
        win32api.mouse_event(win32con.MOUSEEVENTF_LEFTUP, target_x, target_y, 0, 0)
        
        print(f"🦾 [机械臂] 拟人点击完成: ({target_x}, {target_y})")

import time
from PyQt6.QtCore import QObject, pyqtSignal, QTimer

class TokenBucketLimiter(QObject):
    """令牌桶限流器 (防封核心)"""
    # 信号：通知 UI 更新状态 (是否可用, 剩余令牌数)
    ui_state_changed = pyqtSignal(bool, int)

    def init(self, capacity=10, refill_rate=1):
        """
        :param capacity: 桶容量 (例如：最多连续操作 10 次)
        :param refill_rate: 每秒补充的令牌数 (例如：每秒恢复 1 次操作额度)
        """
        super().init()
        self.capacity = capacity
        self.tokens = capacity
        self.refill_rate = refill_rate
        self.last_refill = time.time()
        self.is_locked = False
        
        # 启动定时器，每秒刷新令牌并通知 UI
        self.timer = QTimer(self)
        self.timer.timeout.connect(self._refill_and_notify)
        self.timer.start(1000)

    def consume(self, tokens_needed=1) -> bool:
        """尝试消费令牌 (RPA 每次操作前必须调用)"""
        self._refill()
        if self.tokens >= tokens_needed:
            self.tokens -= tokens_needed
            self._notify_ui()
            return True
        else:
            # 令牌不足，触发熔断锁定
            self.is_locked = True
            self._notify_ui()
            print("🛑 [风控] 操作频率过高，触发防封熔断！系统强制冷却中...")
            return False

    def _refill(self):
        """补充令牌"""
        now = time.time()
        delta = now - self.last_refill
        self.tokens = min(self.capacity, self.tokens + delta * self.refill_rate)
        self.last_refill = now
        
        if self.is_locked and self.tokens >= self.capacity * 0.5:
            self.is_locked = False # 恢复一半额度时解锁

    def _refill_and_notify(self):
        self._refill()
        self._notify_ui()

    def _notify_ui(self):
        """发射信号给 UI"""
        can_operate = not self.is_locked and self.tokens > 0
        self.ui_state_changed.emit(can_operate, int(self.tokens))

import sqlite3
import time
import queue
from PyQt6.QtCore import QThread
from core.event_bus import event_bus

class DataSink(QThread):
    """数据沉淀黑洞 (异步写入 SQLite，绝不阻塞主线程)"""
    def init(self, db_path="waosx_data.db"):
        super().init()
        self.db_path = db_path
        self.data_queue = queue.Queue()
        self.running = True
        self._init_db()

        # 监听所有核心业务事件，扔进队列
        event_bus.ilink_message_received.connect(lambda *args: self._log_event("LEFT_BRAIN_MSG", args))
        event_bus.vision_ui_element_found.connect(lambda *args: self._log_event("RIGHT_BRAIN_VISION", args))
        event_bus.cmd_click_coordinate.connect(lambda *args: self._log_event("RPA_CLICK", args))

    def _init_db(self):
        """初始化本地数据库表"""
        conn = sqlite3.connect(self.db_path)
        cursor = conn.cursor()
        cursor.execute('''
            CREATE TABLE IF NOT EXISTS event_logs (
                id INTEGER PRIMARY KEY AUTOINCREMENT,
                timestamp REAL,
                event_type TEXT,
                payload TEXT
            )
        ''')
        conn.commit()
        conn.close()

    def _log_event(self, event_type, payload):
        """非阻塞入队"""
        self.data_queue.put((time.time(), event_type, str(payload)))

    def run(self):
        """后台死循环，批量写入数据库"""
        print("💾 [数据黑洞] SQLite 异步沉淀引擎已启动...")
        conn = sqlite3.connect(self.db_path)
        cursor = conn.cursor()
        
        while self.running:
            batch = []
            try:
                # 每次最多取 50 条，批量插入提升性能
                for _ in range(50):
                    batch.append(self.data_queue.get_nowait())
            except queue.Empty:
                pass

            if batch:
                cursor.executemany(
                    "INSERT INTO event_logs (timestamp, event_type, payload) VALUES (?, ?, ?)", 
                    batch
                )
                conn.commit()
                
            time.sleep(0.5) # 每 0.5 秒刷盘一次
            
        conn.close()

    def stop(self):
        self.running = False

核心资产保护：机器码绑定与 RSA 授权验证 (core/license_manager.py)绝不能让客户把软件拷给同行！我们采用“机器指纹 + RSA 非对称加密”方案。客户必须提供机器码，您在后台生成 License 文件发给他，他才能用。(前置条件：pip install pycryptodome)

core/license_manager.py
import os
import json
import wmi # pip install wmi
from Crypto.PublicKey import RSA
from Crypto.Signature import pkcs1_15
from Crypto.Hash import SHA256
from PyQt6.QtWidgets import QMessageBox

class LicenseManager:
    """商业级防盗版授权管理器"""
    
    # 您的私钥 (用于您在后台生成 License，绝对不能泄露！)
    # 这里用占位符，实际开发时请自己生成一对 RSA 密钥
    PRIVATE_KEY_PEM = """
    -----BEGIN RSA PRIVATE KEY-----
    YOUR_PRIVATE_KEY_HERE
    -----END RSA PRIVATE KEY-----
    """
    
    # 公钥 (打包进 exe 里，用于验证 License)
    PUBLIC_KEY_PEM = """
    -----BEGIN PUBLIC KEY-----
    YOUR_PUBLIC_KEY_HERE
    -----END PUBLIC KEY-----
    """

    @staticmethod
    def get_machine_code() -> str:
        """获取电脑的唯一机器码 (CPU序列号 + 主板序列号)"""
        try:
            c = wmi.WMI()
            cpu_id = c.Win32_Processor()[0].ProcessorId.strip()
            board_id = c.Win32_BaseBoard()[0].SerialNumber.strip()
            raw_str = f"{cpu_id}_{board_id}"
            # 取 MD5 作为最终机器码，防止明文暴露
            import hashlib
            return hashlib.md5(raw_str.encode('utf-8')).hexdigest()[:16].upper()
        except Exception as e:
            print(f"⚠️ 获取机器码失败: {e}")
            return "UNKNOWN_MACHINE"

    @staticmethod
    def verify_license(license_file_path: str = "license.key") -> bool:
        """验证 License 文件是否合法且匹配当前机器"""
        if not os.path.exists(license_file_path):
            return False

        try:
            with open(license_file_path, "r") as f:
                data = json.load(f)
            
            machine_code = data.get("machine_code")
            signature = bytes.fromhex(data.get("signature"))
            
            # 1. 检查机器码是否匹配当前电脑
            if machine_code != LicenseManager.get_machine_code():
                print("❌ [授权] 机器码不匹配！拒绝运行。")
                return False

            # 2. 使用公钥验证签名 (防止客户自己篡改 license.key)
            key = RSA.import_key(LicenseManager.PUBLIC_KEY_PEM)
            h = SHA256.new(machine_code.encode('utf-8'))
            
            pkcs1_15.new(key).verify(h, signature)
            print("✅ [授权] License 验证通过，尊享商业版功能！")
            return True
            
        except (ValueError, TypeError, KeyError) as e:
            print(f"❌ [授权] 签名验证失败 (文件被篡改): {e}")
            return False
        except Exception as e:
            print(f"❌ [授权] 未知错误: {e}")
            return False

    @staticmethod
    def generate_license_for_client(machine_code: str, output_path: str):
        """【老板专用】在您的电脑上运行此方法，为客户生成 license.key"""
        key = RSA.import_key(LicenseManager.PRIVATE_KEY_PEM)
        h = SHA256.new(machine_code.encode('utf-8'))
        signature = pkcs1_15.new(key).sign(h)
        
        data = {
            "machine_code": machine_code,
            "signature": signature.hex()
        }
        with open(output_path, "w") as f:
            json.dump(data, f)
        print(f"🎉 已成功为客户 [{machine_code}] 生成授权文件: {output_path}")

import multiprocessing
from core.left_brain_real import LeftBrainReal
from core.right_brain_real import RightBrainReal

class MatrixWorker:
    """单账号工作节点 (在独立进程中运行，防止一个微信崩溃拖垮全局)"""
    def init(self, wx_hash: str, wechat_hwnd: int):
        self.wx_hash = wx_hash
        self.wechat_hwnd = wechat_hwnd
        self.left_brain = None
        self.right_brain = None

    def start(self):
        print(f"🚀 [矩阵] 节点 {self.wx_hash} 启动，绑定窗口句柄 {self.wechat_hwnd}")
        # 在子进程中初始化左右脑，彻底隔离内存
        self.left_brain = LeftBrainReal(self.wx_hash)
        self.right_brain = RightBrainReal(self.wechat_hwnd)
        
        self.left_brain.start()
        self.right_brain.start()

    def stop(self):
        if self.left_brain: self.left_brain.stop()
        if self.right_brain: self.right_brain.stop()

class MatrixManager:
    """全局矩阵调度器 (管理多个 Worker)"""
    def init(self):
        self.workers = {} # {wx_hash: multiprocessing.Process}

    def add_account(self, wx_hash: str, wechat_hwnd: int):
        if wx_hash in self.workers:
            return
        
        worker = MatrixWorker(wx_hash, wechat_hwnd)
        # 使用多进程隔离，这是防内存泄漏和防连环崩溃的核心
        p = multiprocessing.Process(target=worker.start)
        p.start()
        self.workers[wx_hash] = p
        print(f"✅ [矩阵] 成功挂载账号: {wx_hash} (PID: {p.pid})")

    def remove_account(self, wx_hash: str):
        if wx_hash in self.workers:
            self.workers[wx_hash].terminate()
            del self.workers[wx_hash]
            print(f"🛑 [矩阵] 已卸载账号: {wx_hash}")

core/watchdog.py
import time
from PyQt6.QtCore import QThread
from core.event_bus import event_bus
from core.matrix_manager import MatrixManager

class Watchdog(QThread):
    """全局看门狗 (系统自愈引擎)"""
    def init(self, matrix_manager: MatrixManager):
        super().init()
        self.matrix = matrix_manager
        self.running = True
        # 记录每个账号最后一次心跳时间
        self.heartbeats = {} 

    def update_heartbeat(self, wx_hash: str):
        """底层模块存活时，持续调用此方法上报心跳"""
        self.heartbeats[wx_hash] = time.time()

    def run(self):
        print("🐕 [看门狗] 全局自愈监控已启动...")
        while self.running:
            now = time.time()
            for wx_hash, process in list(self.matrix.workers.items()):
                # 1. 检查进程是否意外死亡
                if not process.is_alive():
                    print(f"🚨 [看门狗] 警报！账号 {wx_hash} 进程崩溃！准备自愈...")
                    event_bus.ui_show_warning.emit(f"账号 {wx_hash} 崩溃，正在自动重启...")
                    # 这里可以触发自动重启逻辑
                    self._auto_restart(wx_hash)
                    continue

                # 2. 检查心跳超时 (例如超过 30 秒没收到左脑的消息，说明底层卡死了)
                last_beat = self.heartbeats.get(wx_hash, now)
                if now - last_beat > 30:
                    print(f"🚨 [看门狗] 警报！账号 {wx_hash} 假死 (心跳超时)！强制重启...")
                    process.terminate() # 杀掉卡死的进程
                    self._auto_restart(wx_hash)

            time.sleep(5) # 每 5 秒巡检一次

    def _auto_restart(self, wx_hash: str):
        """自愈重启逻辑"""
        # 实际业务中，这里需要重新获取窗口句柄并调用 matrix.add_account
        print(f"🔄 [看门狗] 正在重新拉起 {wx_hash}...")

            parent_widget, "导出运营战报", f"WAOSX_Report_{datetime.now().strftime('%Y%m%d')}.csv", 
            "CSV Files (*.csv)"
        )

            return

            conn = sqlite3.connect(db_path)
            cursor = conn.cursor()

                SELECT event_type, COUNT(*) 
                FROM event_logs 
                WHERE timestamp >= strftime('%s', 'now', '-7 days') 
                GROUP BY event_type
            """)
            data = cursor.fetchall()
            conn.close()

            with open(file_path, 'w', newline='', encoding='utf-8-sig') as f:
                writer = csv.writer(f)
                writer.writerow(["事件类型", "本周触发次数"])
                writer.writerows(data)
                
            print(f"✅ [运营] 战报已成功导出至: {file_path}")
